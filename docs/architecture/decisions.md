# 核心决策

已确定，不可更改。

| # | 决策项 | 结论 |
|---|--------|------|
| 1 | 决策模型 | FSM + BT 混合，FSM 管宏观状态转换，BT 管状态内行为 |
| 2 | 配置存储 | 开发 JSON + 生产 MongoDB，Loader 抽象数据源 |
| 3 | 条件表达式 | 简单规则匹配器，结构化 JSON，AND/OR 组合 |
| 4 | Blackboard | 泛型 `BBKey[T]` + 注册表校验 + 集中 const 文件 |
| 5 | 决策中心 | 事件分发 + 威胁评估 + 优先级仲裁 |
| 6 | 实验框架 | build tag 隔离，必须有对照数据（Hybrid vs PureFSM vs PureBT） |
| 7 | NPC 类型注册 | 注册表/工厂模式 |

## 配置必须覆盖的组件

NPC 类型参数、性格参数、FSM 状态+转换规则、BT 树结构、事件类型定义（含感知方式）、NPC 生成点——全部从配置加载，禁止硬编码。
